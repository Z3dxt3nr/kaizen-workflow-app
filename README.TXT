Kaizen Workflow Lite v1.3.0
===========================

Thank you for downloading Kaizen Workflow — a complete Kaizen, Lean, and 5S improvement tracking app.

📦 Contents:
- /public — main app files (HTML, CSS, JS)
- server.js — lightweight local server
- version.json — app version and edition info

⚙️ How to Run:
1. Make sure you have Node.js installed (https://nodejs.org)
2. Open a terminal in this folder
3. Type:
   npm install
   node server.js
4. Open your browser and go to:
   http://localhost:4000

🧰 Features:
- Idea Tracker with photo upload & reset
- Dashboard with live metrics & monthly charts
- 5S Audit Tool (Tools > 5S)
- Local storage (no internet required)
- Works on Windows, Mac, or Linux

📁 Resetting Data:
Click the “Reset All Data” button on the Tracker page.

🚀 Upgrading:
Future updates can be imported by replacing files and keeping version.json for reference.
